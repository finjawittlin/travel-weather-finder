# Travel Weather Finder

## Projektübersicht

Travel Weather Finder ist eine responsive Web-Applikation, die Nutzern passende Reisedestinationen basierend auf ihren bevorzugten Wetterbedingungen vorschlägt.

Anstatt selbst nach einem Reiseziel zu suchen, können Nutzer eine Wetterpräferenz wie **Sunny**, **Tropical**, **Rainy**, **Cloudy** oder **Cold** auswählen. Die Anwendung ruft anschliessend aktuelle Wetterdaten über die Open-Meteo API ab und schlägt eine passende Destination vor.

Das Projekt wurde im Rahmen des Moduls **Interaktive Medien II)** an der FHGR entwickelt.

---

## Funktionen

- Wetterbasierte Reisezielsuche
- Abruf von Echtzeit-Wetterdaten über die Open-Meteo API
- Dynamische Destinationsempfehlungen
- Auswertung der Wettervorhersage für die nächsten sieben Tage
- Anzeige von Wetterinformationen und Wetter-Icons
- Quick Facts zu jeder Destination
- Interaktive Ladeanimation mit Lottie
- Responsives Design für Desktop und Mobile
- Benutzerfreundliches Custom-Dropdown
- Fehlerbehandlung bei fehlender Wetterauswahl

## API

Für dieses Projekt wird die Open-Meteo API verwendet:

https://www.freepublicapis.com/open-meteo

Folgende Wetterdaten werden abgerufen:

- Wetterzustand
- Temperaturvorhersage
- Windgeschwindigkeit
- Niederschlagsmenge

Die Daten werden anschliessend verarbeitet und in Wetterkategorien eingeteilt, um passende Reisedestinationen vorzuschlagen.

---

## Benutzerablauf

1. Wetterpräferenz auswählen.
2. Auf „Find a destination“ klicken.
3. Die Anwendung ruft aktuelle Wetterdaten ab.
4. Eine passende Reisedestination wird vorgeschlagen.
5. Über „Explore“ können weitere Informationen zur Destination angezeigt werden.
6. Anschliessend kann jederzeit eine neue Suche gestartet werden.
